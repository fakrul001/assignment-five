What is Document Object Model (DOM)

Dom, Document Object Model is a programming interface that represents structured documents like HTML as a kind of object tree. It allows the user to create, remove, modify the elements of the document.

how do you select an element from the DOM

There are five methods to select an element from the DOM. They are:

  • Using getElementById
  • Using getElementByClassName
  • Using getElementByTagName
  • Using querySelector
  • Using querySelectorAll
What is event delegation in the context of the dom, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

UsefulNess of event delegation is given below:

  • Reduces memory usage and improves overall performance.
  • Makes the code more maintainable and easier to update.
  • No need to manually attach or remove event listeners when the DOM structure changes
